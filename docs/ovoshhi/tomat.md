# Томат 'Бычье сердце' 🍅

## 🌱 Характеристики сорта и использование
- **Тип:** Индетерминантный (высокорослый)
- **Срок созревания:** Среднеспелый (110-120 дней)
- **Назначение:** Салатный, для свежего потребления
- **Цвет плодов:** Малиновый
- **Масса плода:** 300-500 г
- **Урожайность:** Высокая (до 5 кг с куста)

## 🗓️ Календарь работ
Этот раздел удобно оформить таблицей Markdown.

| Месяц           | Работы                                                        |
| --------------- | ------------------------------------------------------------- |
| Март            | Посев семян на рассаду (за 60-65 дней до высадки). Пикировка. |
| Апрель-Май      | Закаливание рассады.                                          |
| Май-Июнь        | Высадка в грунт (после угрозы заморозков).                    |
| Июнь-Август     | Полив, подкормки, пасынкование, подвязка.                     |
| Август-Сентябрь | Сбор урожая. Удаление верхушек для остановки роста.           |

## 📅 Пошаговая агротехника
1.  **Подготовка семян:** Замачивание в стимуляторе роста на 12 часов.
2.  **Посев:** Глубина заделки 1-1.5 см.
3.  **Уход за рассадой:** Температура +23-25°C днем.
4.  **Высадка в грунт:** Схема посадки 50x60 см.
5.  **Формирование куста:** Обязательно вести в 2 стебля.

## 💡 Важные нюансы для региона и сорта
> [!tip] Важно для Средней полосы
> Из-за крупных плодов кисти нужно подвязывать индивидуально, иначе ветки могут обломиться.

> [!warning] Особенность сорта
> 'Бычье сердце' склонно к растрескиванию при неравномерном поливе. Поливайте регулярно, но умеренно.
