# Спаржа ‘Аржентельская’ 🌱

## 🌱 Характеристики сорта и использование
- **Тип растения:** многолетняя овощная культура (аспарагус лекарственный, *Asparagus officinalis*)
- **Срок вступления в плодоношение:** 2–3 год после посадки (полноценный урожай с 3 года)
- **Назначение:** деликатесная овощная культура (отваривание, запекание, гриль, заморозка)
- **Цвет побегов:** белый (при окучивании), зелёно-фиолетовый (без отбеливания)
- **Форма побегов:** толстые, прямые, цилиндрические «копья»
- **Длина побегов:** 15–25 см
- **Диаметр побегов:** до 2–3 см (крупные, мясистые)
- **Вкус:** нежный, слегка сладковатый, с ореховыми нотами
- **Урожайность:** высокая при зрелых посадках (до 3–4 кг/м² в сезон)
- **Долговечность плантации:** 10–15+ лет на одном месте
- **Особенность сорта:** классический французский сорт из Аржантёя, известен как один из лучших для белой спаржи, ценится за раннее отрастание и нежную текстуру побегов

| Февраль–март        | Посев семян на рассаду (или подготовка гряд под посадку)    |
| Апрель–май          | Пикировка, выращивание рассады, уход                        |
| Май–июнь            | Высадка на постоянное место (глубокие гряды, богатая почва) |

| Август              | Накопление питательных веществ, рост корней                 |
| Сентябрь            | Подготовка к зимовке, обрезка надземной части               |
| Октябрь             | Мульчирование, защита на зиму                               |

## 📅 Пошаговая агротехника
1. **Выбор места** — солнечное, защищённое от ветра, с лёгкой дренированной почвой.
2. **Подготовка почвы** — глубокая перекопка (до 30–40 см), внесение органики.
3. **Посев/рассада** — выращивание из семян или посадка корневищ (крон).
4. **Высадка** — в глубокие борозды (15–25 см), с постепенным засыпанием.
5. **Уход в первый год** — только наращивание корней, без сбора урожая.
6. **Окучивание (отбеливание)** — формирование белых нежных побегов.
7. **Полив и питание** — регулярный, особенно в период активного роста.
8. **Сбор урожая** — весной, срезка молодых побегов до их огрубения.

## 💡 Важные нюансы для региона и сорта

!!! tip "Красноярский край (пригород Красноярска)"
    Сорт хорошо зимует, но требует обязательного мульчирования (торф, перегной, солома) для защиты корней от промерзания.

!!! warning "Долгий старт"
    Спаржа 'Аржентельская' требует терпения: полноценный урожай начинается только через 2–3 года после посадки.

!!! tip "Плюс сорта"
    Один из лучших классических сортов для белой спаржи — даёт толстые, нежные и вкусные побеги при правильном окучивании.

!!! warning "Ошибки выращивания"
    - нельзя срезать урожай в первый год
    - нельзя пересушивать почву в период роста
    - нельзя уплотнять почву (спаржа любит воздух в корнях)
