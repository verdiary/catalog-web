# Редис 'Кормилец' 🥗

## 🌱 Характеристики сорта и использование
- **Тип:** Однолетняя овощная культура (редис, *Raphanus sativus*)
- **Срок созревания:** Очень ранний (18–20 дней от всходов)
- **Условия выращивания:** Открытый грунт, возможен посев в теплицах
- **Форма корнеплода:** Удлинённая (цилиндрическая)
- **Цвет:** *требуется уточнение для данного сорта* (вероятно красный с белой мякотью, типично для культуры)  
- **Масса корнеплода:** 20–23 г
- **Мякоть:** Белая, сочная, хрустящая, слабоострая
- **Урожайность:** 2,5–3 кг/м²
- **Назначение:** Свежие салаты, окрошка, ранняя витаминная продукция  
- **Устойчивость:** Устойчив к стрелкованию и пустотелости мякоти
- **Особенности:** Не дряблеет при перерастании, подходит для конвейерного выращивания

## 🗓️ Календарь работ
| Месяц    | Работы                                                            |
| -------- | ----------------------------------------------------------------- |
| Апрель   | Первый посев в грунт под укрытие (при прогреве почвы от +5…+8°C). |
| Май      | Повторные посевы каждые 15–20 дней, прореживание всходов.         |
| Июнь     | Регулярные поливы, рыхление, сбор первого урожая.                 |
| Июль     | Повторные посевы (при необходимости), защита от жары.             |
| Август   | Посев для осеннего урожая, поддержание влажности.                 |
| Сентябрь | Сбор позднего урожая.                                             |

## 📅 Пошаговая агротехника
1. **Подготовка почвы:** Лёгкая, рыхлая, плодородная почва с нейтральной реакцией (pH 6–7).  
2. **Подготовка семян:** Замачивание на 8–12 часов (по желанию) для ускорения всходов.  
3. **Посев:**  
   - Глубина: 1–1,5 см  
   - Схема: 5×10 см
4. **Температура:** Всходы появляются уже при +5…+8°C, оптимум +15…+20°C.  
5. **Прореживание:** После появления 1–2 настоящих листьев оставить расстояние 4–5 см.  
6. **Полив:** Регулярный и равномерный — при пересыхании почвы корнеплоды грубеют.  
7. **Подкормки:** Обычно не требуются при хорошо заправленной почве (*требуется уточнение для данного сорта*).  
8. **Сбор урожая:** Через 18–20 дней, выборочно по мере достижения размера.  

## 💡 Важные нюансы для региона и сорта

!!! tip "Красноярский край (Сибирь)"
    Весна затяжная и холодная — используйте укрытие (агроволокно или плёнку) для ранних посевов. Это ускорит развитие и защитит от возвратных заморозков.

!!! tip "Конвейерный посев"
    Сорт отлично подходит для посева каждые 2–3 недели — это позволяет получать стабильный урожай всё лето.

!!! warning "Перепады температуры"
    При резких колебаниях температуры и длинном световом дне возможна стрелкование, несмотря на устойчивость сорта — особенно в июне.

!!! warning "Полив"
    Нерегулярный полив приводит к грубости мякоти и появлению горечи. В условиях сухого климата Красноярского края это критично.

!!! tip "Использование в смешанных посадках"
    Можно выращивать как уплотняющую культуру между капустой, огурцами или томатами — быстро освобождает место для основной культуры.
