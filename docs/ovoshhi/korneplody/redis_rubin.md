# Редис 'Рубин' ❤️

## 🌱 Характеристики сорта и использование
- **Тип:** Однолетняя корнеплодная культура (редис)
- **Срок созревания:** Раннеспелый (18–30 дней от всходов)
- **Назначение:** Салатный, для свежего потребления
- **Форма корнеплода:** Округлая или округло-овальная
- **Цвет:** Рубиново-красный (кожица), мякоть белая или бело-розовая
- **Размер:** Диаметр 3–5 см
- **Масса:** 11–28 г
- **Вкус:** Сочный, нежный, слабоострый, слегка сладковатый
- **Урожайность:** 1,2–2,5 кг/м²
- **Особенности:**
  - Дружная отдача урожая
  - Подходит для открытого и защищённого грунта
  - Относительно устойчив к цветушности и дряблению

## 🗓️ Календарь работ в виде таблицы
| Месяц        | Работы                                                         |
| ------------ | -------------------------------------------------------------- |
| Апрель       | Первый посев в открытый грунт под укрытие или в теплицу        |
| Май          | Основные посевы в открытый грунт, прореживание всходов         |
| Июнь         | Повторные посевы (для конвейера), полив, рыхление              |
| Июль         | Посев (при притенении), регулярный полив                       |
| Август       | Последние посевы, уход за растениями                           |
| Май–Сентябрь | Сбор урожая по мере созревания (через 3–4 недели после посева) |

## 📅 Пошаговая агротехника
1. **Подготовка семян:**  
   Отобрать крупные семена, можно замочить на 6–12 часов для ускорения всходов.

2. **Подготовка почвы:**  
   Лёгкая, плодородная, супесчаная почва с pH 5.5–7.0

3. **Посев:**  
   - Глубина: 1–1,5 см
   - Схема: 5×10 см

4. **Прореживание:**  
   После появления 1–2 настоящих листьев оставить расстояние 4–5 см

5. **Полив:**  
   Регулярный, без пересыхания — редис чувствителен к влаге

6. **Уход:**  
   Рыхление почвы, удаление сорняков

7. **Подкормки:**  
   Обычно не требуются при хорошей почве (*требуется уточнение для данного сорта*)

8. **Сбор урожая:**  
   Через 20–30 дней, не передерживать в почве

## 💡 Важные нюансы для региона и сорта

!!! tip "Для Красноярского края"
    - Начинайте посевы позже, чем в средней полосе — обычно с конца апреля или начала мая
    - Используйте укрывной материал (агроволокно) для защиты от возвратных заморозков
    - Оптимальная температура для роста +15…+18°C, при жаре качество ухудшается

!!! warning "Особенности сорта 'Рубин'"
    - Склонен к **быстрому дряблению при перерастании**, убирайте вовремя
    - При недостатке влаги корнеплоды становятся грубыми и острыми
    - При длинном световом дне возможна стрелкование — летом лучше притенять

!!! tip "Практика для стабильного урожая"
    - Сейте каждые 10–14 дней для непрерывного получения продукции
    - Не загущайте посевы — это снижает размер корнеплодов
