## Как пользоваться справочником

Каждая страница о растении содержит следующие разделы:
- **Характеристики сорта и использование** - основная информация о растении
- **Календарь работ** - таблица с сезонными работами
- **Пошаговая агротехника** - подробные инструкции по выращиванию
- **Важные нюансы** - советы и предупреждения для конкретного региона и сорта
